# 3D Engine in C/C++
![](images/preview.png)
![](images/preview2.png)
## Install dependencies 
- GLM: https://github.com/g-truc/glm?tab=readme-ov-file#build-and-install
## Build
```
$ cmake -B build
$ cmake --build build 
```
## Build in debug mode
```
$ cmake -B build -DDEBUG=ON
$ cmake --build build 
```
## Run 
```
$ ./engine
```
